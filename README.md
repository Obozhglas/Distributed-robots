# Distributed-robots
## Децентрализованная отказоустойчивая система кооперации манипуляторов с распределением задач и обнаружением отказов

### Описание проекта
Данный проект моделирует работу группы роботов, которые совместно обрабатывают поток входящих задач.

### Основные возможности
- Параллельная работа роботов — все роботы выполняют задачи одновременно
- Балансировка нагрузки — задачи назначаются на наименее загруженного робота
- Моделирование отказов — можно вручную отключать роботов
- Автоматическое перераспределение задач при сбоях
- Интерактивный интерфейс (Tkinter)
- Визуализация движения задач к роботам

### Логика работы системы
Система использует простую, но эффективную стратегию распределения: у каждого робота есть показатель нагрузки (load), новая задача назначается роботу с минимальной нагрузкой. При отказе робота его задачи перераспределяются. Все роботы работают параллельно без центрального управляющего узла
